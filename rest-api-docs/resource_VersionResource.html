System Version Resource

Retrieves the system's version.

GET /version

Retrieve Version information for the application.

Response Codes
code condition
200 Response with the system's version information in the body
500 If there is an unexpected error in the server while performing this operation
Response Body
media type data type
application/vnd.zanata.Version+json Version Info (JSON)
application/vnd.zanata.Version+xml Version Info (XML)

Example

Request
GET /version
Accept: application/vnd.zanata.Version+json
Response
HTTP/1.1 200 OK
Content-Type: application/vnd.zanata.Version+json

                
{
  "versionNo" : "4.0.0",
  "buildTimeStamp" : "20170225-1448",
  "scmDescribe" : "..."
}
