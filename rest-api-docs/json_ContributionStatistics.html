Contribution Statistics Data Type

Contains information about translations contributed by a single user.

Properties
name data type description
username string User name responsible for the contributions.
contributions array of Locale Statistics

Example

{
  "username" : "bart",
  "contributions" : [ {
    "locale" : "es-ES",
    "translation-stats" : {
      "approved" : 12345,
      "rejected" : 12345,
      "translated" : 12345,
      "needReview" : 12345
    },
    "review-stats" : {
      "approved" : 12345,
      "rejected" : 12345,
      "translated" : 12345,
      "needReview" : 12345
    }
  }, {
    "locale" : "ja",
